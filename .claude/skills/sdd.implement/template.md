# Template
```markdown
**Created at**: [DATE]
**Based on plan**: @link_path_to_plan
**Based on story**: @link_path_to_story

# Resume: [FEATURE NAME]

### Executive Summary
[2-3 sentences — what was accomplished, business value. No jargon.]

### Technical Summary
- [Key technical change 1]
- [Key technical change 2]
- [Key technical change 3]

### Phases Completed
- [x] **Phase 1**: [Name] — [one-line summary]
- [x] **Phase 2**: [Name] — [one-line summary]
```

# Rules
- **Executive Summary**: 2-3 sentences max. Business value. Accessible to non-technical reviewers.
- **Technical Summary**: 5-7 bullets. Key decisions, files modified, breaking changes, new dependencies.
- **Phases**: Checked checkboxes + one-line summary per phase.
- **Language**: Write in the **same language the user used**.
